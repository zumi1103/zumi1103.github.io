# 📁 My Dev Tools マニュアル

Chrome拡張機能自作やバイブコーディングの効率を極限まで高めるための、個人開発者向け統合ポータルサイトです。
フォルダ内のソースコードを1つのXML（`<repository>`）形式にまとめ、Geminiアプリへ一瞬で共有できる機能を中心に構成されています。

---

## 🛠️ 主な機能

1. **📁 My Repomix (コードおまとめツール)**
   - 開発フォルダをドラッグ＆ドロップ（または選択）するだけで、中のソースコードを1つのテキストに自動集約します。
   - `.git` や `node_modules`、各種画像ファイルなどの不要なファイルは自動で除外されます。
   - ボタン1つでクリップボードにコピーされ、同時にGeminiアプリ（新しいタブ）が立ち上がります。

2. **🎨 My Icon (外部連携 ↗)**
   - Chrome拡張機能開発に必須な各種サイズ（16px, 48px, 128px等）のアイコンを自動生成する外部サービスへ素早くアクセスできます。

3. **📝 My Editor (外部連携 ↗)**
   - ご自身で開発された「My Editor」へスムーズにアクセスし、コードの編集や確認を行えます。

4. **🌙 ダークモード・ライトモード切り替え**
   - 画面右上の ☀️/🌙 ボタンで、開発環境に合わせたテーマの切り替えが可能です。

---

## 🚀 使い方（My Repomix の基本フロー）

### ステップ 1: フォルダの読み込み
「📁 My Repomix」の画面を開き、集約したい開発フォルダ（Chrome拡張機能のフォルダなど）を **「ここにフォルダをドラッグ＆ドロップ」** のエリアへドラッグ＆ドロップします。
※ または「フォルダを選択」ボタンからでも読み込めます。

### ステップ 2: コピー & Geminiの起動
フォルダの解析が終わると、下部のテキストエリアに集約されたコードが表示され、上部のボタンが有効化されます。
**「📋 結果をコピーしてGeminiを開く」** ボタンをクリックします。

### ステップ 3: Geminiへペーストしてプロンプト入力
自動的に新しいタブでGeminiが開くので、メッセージ入力欄に `Ctrl + V`（Macは `Cmd + V`）でコピーした内容を貼り付けます。
その後、以下のようなプロンプトを入力して「バイブコーディング」を開始します。

> **プロンプトの例:**
> 「これが現在のChrome拡張機能のソースコード一式です。新しく〇〇の機能を追加したいので、修正コードを提示してください。」

## 📂 画面右上・ヘッダーアイコンについて

- **📁 フォルダマーク**: 
  クリックすると、このサイトのソースコードが管理されているGitHubリポジトリ（`zumi1103.github.io`）へ直接移動します。コードの更新やコミットの際に使用してください。
- **☀️ / 🌙 マーク**: 
  デザインのテーマ（ダークモード/ライトモード）をワンクリックで切り替えます。
