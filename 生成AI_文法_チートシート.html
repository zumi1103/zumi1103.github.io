AIプロンプト作成チートシート

効果的な記号と記法ガイド

なぜ記号を使うのか？

AIへの指示（プロンプト）で記号を使うと、人間が文章の構成を考えるのと同じように、AIが指示の構造や意図を正確に理解できるようになります。

  • 指示が明確になる：命令、参考情報、制約条件などを区別できる。
  • 出力の質が上がる：より意図に沿った、精度の高い回答が返ってくる。
  • 複雑な命令が可能になる：手順を踏んだり、特定の形式で出力させたりできる。

`#` (見出し)

役割：プロンプトの構造化

`#` を使うと、文章のセクションを分けることができます。AIに対して、指示の全体像と各部分の役割を明確に伝えます。

# 役割
あなたはプロのマーケターです。

## 指示
以下の新商品のキャッチコピーを3つ提案してください。

### 制約条件
- ターゲットは20代の若者
- 短く、覚えやすいフレーズにする

## 商品情報
- 商品名：スマートイヤホン「OtoLink」
- 特徴：高音質、長時間バッテリー、防水機能

`*` (箇条書き / 強調)

役割：情報の整理と重要点の伝達

`*` は、項目をリスト化したり、特定の単語を強調したりするのに便利です。

来週のチームミーティングのアジェンダを作成してください。
含めるべき項目は以下の通りです。

* 先週の進捗確認
* **今週の課題共有**
* 新規プロジェクトのブレインストーミング

特に `**課題共有**` の部分に時間を多めに割く構成にしてください。

`>` (引用ブロック)

役割：指示文と処理対象テキストの分離

`>` を使うことで、あなたの「指示」と、AIに「処理してほしい文章」を明確に区別できます。

以下のカスタマーレビューを要約し、ポジティブな点とネガティブな点を抽出してください。

> このカメラは画質が本当に素晴らしいです。特に夜景モードは感動しました。
> ただ、バッテリーの持ちが少し短いのが残念なポイントです。
> 全体的には満足しています。

`` ` `` (コードブロック)

役割：コードや特定の出力フォーマットを指定

バッククォート3つで囲むと、その中のテキストはAIが特定の形式を持つデータとして正確に認識します。

例1：コードの記述

以下のPythonコードに、処理内容を説明するコメントを追加してください。
```python
def fibonacci(n):
    a, b = 0, 1
    while a < n:
        print(a, end=' ')
        a, b = b, a+b
```

例2：出力形式の指定

私の自己紹介文を、以下のJSON形式で生成してください。
```json
{
  "name": "山田 太郎",
  "occupation": "ソフトウェアエンジニア",
  "hobby": ["読書", "旅行"]
}
```

`` (XMLタグ)

役割：日本語の「」のような記号。各単語や要素の役割を厳密に定義

`#` よりもさらに厳密にセクションの役割を伝えたい場合に効果的です。AIはXML/HTML形式の構造理解に長けています。


あなたは経験豊富な旅行プランナーです。



以下のに基づいて、3日間の北海道旅行プランを提案してください。



- 時期：7月
- 興味：自然、グルメ
- 予算：1人10万円



1日ごとのスケジュール形式で回答してください。

`1. 2.` (番号付きリスト)

役割：手順や思考プロセスの指定

AIに特定の順序でタスクを実行させたり、段階的に思考させたりしたい場合に使います。

「リモートワークの未来」というテーマでブログ記事の構成案を作成してください。
以下の手順で思考してください。

1. まず、リモートワークの現状のメリットとデメリットを洗い出す。
2. 次に、テクノロジーの進化がそれをどう変えるかを予測する。
3. 最後に、未来の理想的な働き方について結論をまとめる。

`[] {}` (括弧)

役割：変数やデータ構造の指定

プロンプトをテンプレート化したり、複雑なデータ形式を扱ったりする際に役立ちます。

例1：テンプレート化 `[]`

以下のテンプレートを使って、商品の説明文を作成してください。

商品名：[商品名]
ターゲット：[ターゲット層]
伝えたい魅力：[商品の魅力]

例2：データ構造の指定 `{}`

日本の主要都市について、以下の形式のオブジェクトを3つ作成してください。
{ "city": "都市名", "population": "人口", "specialty": "名物" }

'{}' '...' (プレースホルダー)

役割：変数入力と任意入力の指示

`{}`はユーザーが具体的な内容を補完する変数部分を、`'...'`はAIやユーザーが自由に記述・補完する部分を示すのに使います。これにより、プロンプトのテンプレートをより柔軟に作成できます。

例1：質問内容の指定 `{}`

以下の質問に答えてください。

{ここに質問を入力してください}

例2：自由記述の指示 `'...'`

以下の対話形式のテキストを完成させてください。

ユーザー: こんにちは！今日の天気はどうですか？
AI: こんにちは！ '{ここに今日の天気に関する応答が入ります}'
ユーザー: なるほど、ありがとうございます。 '...'
AI: どういたしまして。 '...'
